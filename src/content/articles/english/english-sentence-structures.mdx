# 英語基本句型詳解

基本句型是一個句子最精簡的結構，這個最精簡的結構是一個完整且語法正確的句子。

詞是句子的最小令牌，詞與詞按照一個合法的排列方式構成句子，按照合法排列方式排列的詞和詞們扮演不同的角色（詞性），詞性是不同性質的詞的抽象層：

| 詞性               | 定義                               |     |
| ---------------- | -------------------------------- | --- |
| Nouns 名詞         | 表示人事物和抽象概念                       |     |
| Pronouns 代名詞     | 代替名詞                             |     |
| Determiner 限定詞   | 修飾名詞，限定名詞的指定對象，通常包括代名詞、冠詞、數詞、量詞等 |     |
| Adjectives 形容詞   | 修飾人事物，表示性質、屬性和狀態等                |     |
| Verbs 動詞         | 表示動作、狀態、行為和變化等                   |     |
| Adverbs 副詞       | 修飾動詞、形容詞和副詞，表示程度、時間、頻率等          |     |
| Preposition 介系詞  | 與名詞、代詞搭配使用，表示地點、方位、方式等           |     |
| Particles 助詞     | 輔助動詞構成更加複雜的語氣、時態、結構              |     |
| Interjections 嘆詞 | 表示感嘆                             |     |
| Onomatopoeia 擬聲詞 | 模擬聲音                             |     |

## 預備知識

### SVOC的角色認知

劃分句子結構時，通常使用「SVOC」等英文字母表示句子中的主要句式結構，例如S表示主詞，V表示動詞，O表示受詞，C表示補語：

| 詞位  | 意義                 |
| --- | ------------------ |
| S   | Subject            |
| V   | Verb               |
| O   | Object             |
| DO  | Direct Object      |
| IO  | Indirect Object    |
| C   | Complement         |
| SC  | Subject Complement |
| OC  | Object Complement  |

#### Subject

Subject表示主詞，是動作的發出者。

#### Object

Object表示受詞，在SVOO中受詞可細分為：

- 直接受詞 Direct Object
- 間接受詞 Indirect Object

在SVOO句型中，第一個O表示間接受詞，第二個O表示直接受詞。大多數情況下請遵循「先人後事」。

#### Verb

Verb表示動詞。通常情況下，基本句型中的V謂詞動詞，在一個完整的句子中最多出現一次。

### Complement

Complement表示補語，經常出現在SVC和SVOC句型中。補語可分為：

- 主詞補語 Subject Complement
- 受詞補語 Object Complement

### 連系動詞

連系動詞表達「是」，經常出現在SVC和SVOC結構中，起到連接作用（將補語表達的狀態、屬性賦予到被補語修飾的對象）。

連系動詞通常包括：

- Be動詞（am, is, are, was, were, be, being, been)
- 感官動詞 (sound, smell, taste, fell, hear, see, ...)
- Keep, make, find, seem, ...

注意，並非所有連系動詞都只能做「連系作用」。例如become，become作為連系動詞表達「成為」「變成」，become作為實意動詞時表達「適合」。也就是說，**一個句子中出現了連系動詞模樣的動詞，這並不意味著這個句子是SVC或SVOC結構，它可能是SVO或SVOO結構**。

## 五個基本句型

_本文章不討論被動語態。在主動語態中，主詞表示動作的發出者，而在被動語態中，主詞表示動作的接受者。_

通常情況下基本句型分為5種:

- S V
- S V O
- S V O O （可視為S V IO DO）
- S V C （可視為S V SC）
- S V O C （可視為S V O OC）

針對這五個基本句型，有一個簡單的表格做出描述：

| 句型   | 是否及物 | 是否完全 |
| ---- | ---- | ---- |
| SV   | 否    | 是    |
| SVO  | 是    | 是    |
| SVOO | 是    | 是    |
| SVC  | 否    | 否    |
| SVOC | 是    | 否    |

### SV & SVO & SVOO

在SV句型中，動詞不及物且完全，這意味著，SV句型中的動詞**需要動作發出著而不需要動作接受者**：

- Birds fly.
- She smiled.
- They arrived.
- The phone rang.

在SVO句型和SVOO句型中，動詞及物且完全，這意味著SVO句型和SVOO句型都**需要動作的發出者和動作的接受者**。SVO句型中的受詞只有一個，而SVOO中的受詞有兩個，第一個O是IO間接受詞，第二個O是DO直接受詞：

- (SVO) I eat pizza.
- (SVO) The dog chased the ball.
- (SVO) They play soccer.
- (SVOO) I gave her a book.
- (SVOO) He told us a story.
- (SVOO) They offered me a job.

一般情況下，像「give」「tell」等動詞可以借用to和其它介詞來轉換SVO與SVOO結構：

- (SVOO) I - gave - her - a book.
- (SVO) I - gave - a book to her.

### SVC & SVOC

*SVC有時會被稱為SVP，P表示表語。*

在SVC句型中，動詞不及物且不完全，這意味著SVC句型**不需要受詞但需要補語**，這個補語修飾主詞，被稱為「主詞補語」。SVC句型中的動詞通常是be動詞「am」「is」「are」「was」「were」「be」等以及感官動詞「sound」「taste」「fell」「smell」等。這些動詞有一個名稱叫「連系動詞」，這些動詞大多情況下表達「是」，作為是連接前後兩者，將補語跨過動詞直接修飾主詞。

- She is a teacher.
- The soup tastes delicious.
- They became good friends.

在SVOC句型中，動詞及物且不完全，這意味著SVOC句型需要一個受詞和一個補語，並且這個補語與受詞構成be的關係，這個補語被稱為「受詞補語」。

- The news made him sad.
- We consider her an expert.
- I found the book interesting.

### 辨析SVO & SVC & SVOO & SVOC

辨析SVO與SVC句型，首先需要做的是找到動詞，並分析動詞是否為連系動詞。如果是連系動詞並且表達「是」起連接作用，則句子是SVC結構，如果不是連系動詞，則句子是SVO結構：

- (SVC) He **became** a doctor.
- (SVO) She **reads** a **book**.

請參考如上兩個句子。「He」「a doctor」通過連系動詞「became」構成be的關係，將「醫生」屬性賦予給「他」。第二個句子的「She」「a book」通過實意動詞「reads」連接，顯然不能構成be關係。

辨析SVOO與SVOC句型，首先需要做的是找到句子的OO/OC部分，判斷OO/OC是否達成be的關係。如果是be的關係，則句子是SVOC結構，如果不是，則句子是SVOO結構：

- (SVOC) We found the movie interesting.
- (SVOO) She gave me a gift.

請參考如上兩個句子。第一個句子的「the movie」「interesting」構成be的關係，即「有趣」屬性賦予給「電影」，「the movie」是受詞，「interesting」是受詞補語。第二個句子顯然無法將「a gift」作為補語修飾「me」。
