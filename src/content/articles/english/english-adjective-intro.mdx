# 淺談英語形容詞與它的比較句法和特殊句式
_本文將從形容詞的句法功能以及形容詞的比較句法的角度淺談形容詞的作用。_

在學習形容詞之前，需要了解形容詞的基本介紹：
形容詞作為修飾語，它可以修飾名詞詞性的人事物，修飾方法是將形容詞放置在被修飾詞的附近（之前或之後），**大多情況下形容詞被放置在被修飾詞之前，描述被修飾詞的屬性、特徵等**。形容詞在修飾一個主體時，形容詞大多時候是可有可無的，即形容詞大多數時候可以被省略。

## 形容詞的句法功能

| 詞性   | 說明                                                                                  |
| ---- | ----------------------------------------------------------------------------------- |
| 前置定語 | 最常見的用法。通常將形容詞放置在名詞詞性之前，構成「修飾詞 + 被修飾詞」結構。                                            |
| 後置定語 | 通常在修飾「something」「anything」「everything」「else」等不定代詞時將形容詞放置在那些不定代詞之後，構成「被修飾詞 + 修飾詞」結構。 |
| 主詞補語 | 在SVC句式結構中修飾主詞。                                                                      |
| 受詞補語 | 在SVOC句式結構中修飾受詞。                                                                     |
## 形容詞片語
形容詞片語是以形容詞為中心的短語：
- 副詞詞位 形容詞詞位 介詞詞位
- 副詞詞位 形容詞詞位 不定式詞位
- 副詞詞位 形容詞詞位
- 副詞詞位 形容詞詞位 子句

**very proud of**.
**eager to learn**.
**very beautiful**.
**sure that** he will come.
### Enough用法
Enough可修飾名詞詞性，修飾名詞詞性時enough可在名詞詞性前也可在後（通常情況下，enough放置在名詞詞性之前）。修飾形容詞時，一般將enough放置在形容詞之後：
**enough food**.
**hot enough**.

### Such和So用法
So通常修飾形容詞，such通常修飾名詞詞性。它們之間存在一個較為合理的轉換方式：
`So ADJ a/an NOUN ≈ Such a/an ADJ NOUN`
## 形容詞的數
**形容詞天生不具備數的變化。** 在保持形容詞意義的情況下，盡可能的使用單數形式。
一些形容詞比較特殊，它們不能修飾所有形式的名詞詞性，例如many/much，few/little：
- much/little修飾不可數名詞
- many/few修飾可數名詞複數
## 形容詞的比較級與最高級
形容詞的比較級形式與最高級形式相比原本的，比較級與最高級有「比較」的意味，其中最高級的比較意味最深，通常可翻譯為「最...」。
### 原級 & 比較級 & 最高級的轉換
通常情況下，比較級與最高級的變換方法是根據形容詞原級的音節區分的。
大多數情況下，將一個形容詞讀出來，若它的音節數為2個左右，它的「比較級/最高級」的形式為「...er/...est」，若音節數大於3個，形式為「more .../the most ...」。
一些特殊情況下，需要根據形容詞的讀音靈活變通：若形容詞結尾是e，則直接加「...r/...st」，若形容詞結尾是y，則將y變為「...ier/...iest」。

|      | 比較級              | 最高級                  |
| ---- | ---------------- | -------------------- |
| 少音節數 | lower            | lowest               |
| 多音節數 | more interesting | the most interesting |
| e結尾  | cuter            | cutest               |
| y結尾  | happier          | happiest             |

### 常見的特殊的比較級與最高級

| 原級         | 比較級            | 最高級               |
| ---------- | -------------- | ----------------- |
| good/well  | better         | best              |
| bad        | worse          | worst             |
| old/eld    | older/elder    | oldest/eldest     |
| far        | father/further | farthest/furthest |
| much/mamy  | more           | most              |
| little/few | less           | least             |

## 形容詞的比較句法
### The more ..., the more ...
表示「越...就越...」：
**The more** the goods cost, **the better** they must be.
**The more diligently** you study, **the fewer** mistakes you will make.
### More ans more ...
表示「越來越...」：
More and more importantly.
More ans more beautiful.
Colder and colder.
### As ... as ...
表示「像...一樣...」，「as...as...」的格式為「... as 形容詞原級 (名詞詞性) as ...」，否定形式為「... not as/so 形容詞原級 (名詞詞性) as ...」：
He is **as tall as his brother**.
There are **as many apples as oranges** in the basket.
The test was **not as difficult as I expected**.

「as...as...」句型可以在一些句型中省略前者：
（完整）He is not as rich as he pretends, **but as clever as he needs to be**.
（省略）He is not as rich as he pretends, **but clever as he needs to be**.
### More than ...
表示「比...更...」：
There were **more than fifty people** at the meeting.
This book costs **more than I expected**.
The value of A is **greater than** the value of B.
My house is **smaller than** yours.

一些特殊的單詞（...or，...able結尾的等，並且自身表示比較意味的形容詞）沒有比較級和最高級，而且在表達「比...更...」時使用「... to ...」：
He is **senior to** me in the company.
This product is **inferior to** the original.
### 比較句法的倍數表達

| 原句型             | 倍數表達                       |
| --------------- | -------------------------- |
| as ... as ...   | twice as ... as ...        |
| more than ...   | three times more than ...  |
| the size of ... | four times the size of ... |
## 形容詞的特殊句式

| 句式                   | 意義                       |
| -------------------- | ------------------------ |
| Too ADJ to do ...    | 太...而不能...               |
| ADJ Enough to do ... | 足夠...做...                |
| The ADJ              | 表示一類人，請參考名詞片語中的名詞片語省略名詞。 |
| So ADJ that ...      | 太...以至於...               |
| It is ADJ to do ...  | ...很...                  |
