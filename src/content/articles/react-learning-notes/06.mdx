# React Learning Day 5: React Redux Part 2

## createSlice - prepare - Preparing Action Payloads

createSlice的reducers對象的元素除了是`reducer`函數，來可以是`{reducer, prepare}`對象。

`prepare`的返回值是`reducer`的action參數。

```typescript
add: {
    prepare: (firstName: string, lastName: string) => {
        return { payload:  firstName + ' ' + lastName}
    },
    reducer: (state, action: PayloadAction<string>) => {
        state.items.push(action.payload)
    },
}
```

當add時，應使用prepare的參數列表:

```typescript
dispatch(add('first-name', 'last-name'))
```

## createSlice - extraReducers - Handle Other Actions

extraReducers接受一個參數為`builder: ActionReducerMapBuilder<State>`的函數。`builder`提供3個方法:

- addMatcher
- addCase
- addDefaultCase
